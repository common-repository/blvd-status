== Description ==

Plugin to add your BLVD Status tracking code to every page within your blog, without modifying any template files. Includes SSL page support.

You must have a BLVD Status account for it to work. You can get your FREE account by signing up <a href="http://blvdstatus.com/users/createAccount.php5">here.</a>

Also, make sure your server is running PHP 5.

BLVD Status is gives you a live snapshot of what is happening on your website. Some features include:

* Live rankings of your keywords 
* RSS subscriber tracking
* Painless conversion setup
* Easily track outgoing links without any coding knowledge
* View your latest referrers and track which perform best

== Installation ==

1. Upload `blvd-status` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Click on the BLVD Status link under settings.
1. Paste your BLVD Status tracking code for that domain in the box provided.
